# 🚀 Projet Flutter Web - BTS SIO

Ce projet est configuré pour être développé exclusivement sous **GitHub Codespaces**. Il utilise une image Docker pré-configurée avec Flutter pour faciliter votre environnement de travail.

---

## 🛠️ Démarrage de l'application

Une fois votre environnement Codespace prêt (lorsque le terminal est disponible) :

1. **Lancer le serveur de développement** :
   Copiez et collez la commande suivante dans le terminal :
   ```bash
   flutter run -d web-server --web-hostname 0.0.0.0 --web-port 8080 --web-renderer html
